Equation of a Straight Line — y = mx + c
Use the sliders or drag the gradient handle and y-intercept dot directly on the graph.
y = x
m (gradient) 1
c (y-intercept) 0
Cambridge tip: read c directly from where the line crosses the y-axis, then pick two integer points to find \(m = \dfrac{y_2-y_1}{x_2-x_1}\). Write the final answer as \(y = mx + c\) — leaving it in \(y - y_1 = m(x - x_1)\) form loses the A1 if the question specifies the form.
